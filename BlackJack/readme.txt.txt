The program contains the Blackjack Game with the following rules.

# The deck is unlimited in size. 
# There are no jokers. 
# The Jack/Queen/King all count as 10.
# The the Ace can count as 11 or 1.
# The cards have equal probability of being drawn.
# Cards are not removed from the deck as they are drawn.
# The computer is the dealer.

The classic version of the game can be found at https://games.washingtonpost.com/games/blackjack.

Replit version.
